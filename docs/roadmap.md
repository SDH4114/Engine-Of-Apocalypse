# Engine Of Apocalypse – План развития (Roadmap)

## Текущий статус: Milestone 1 (Завершен)
- [x] Базовая структура репозитория.
- [x] Интеграция SDL2 и Veldrid.
- [x] Реализация основного цикла (Game Loop).
- [x] Абстракция графического контекста (очистка экрана, swap buffers).
- [x] Кроссплатформенная сборка и запуск (macOS/Windows исправлены и протестированы).
- [x] Пример игры (SampleGame), корректно запускающийся и работающий.

## Предстоящие этапы

### Milestone 2: Время и Диагностика
*   Реализация фиксированного шага времени (fixed timestep) для обновления физики/логики.
*   Экранная диагностика (счетчик FPS, время кадра).
*   Логирование.

### Milestone 3: 2D Рендеринг (База)
*   Загрузка текстур.
*   SpriteBatch: отрисовка 2D спрайтов.
*   Камера и трансформации.

### Milestone 4: Ввод (Input)
*   Абстракция ввода (клавиатура, мышь, геймпады).
*   Привязка действий (Action Mapping).

### Milestone 5: Физика и Ассеты
*   Интеграция физического движка (Box2D/Velcro).
*   Конвейер импорта ассетов.
*   База данных ресурсов.

### Milestone 6: UI и Инструменты
*   Интеграция ImGui для инструментов отладки.
*   Система пользовательского интерфейса (UI) для игры.

### Milestone 7+
*   Аудио (EoA.Audio).
*   Редактор карт/уровней.
*   Синематика.
